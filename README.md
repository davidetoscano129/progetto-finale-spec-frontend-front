# University Faculty Comparator

## Project Overview

This project is a frontend application built with React that allows users to compare different university faculties. The application provides features such as searching, filtering, sorting, and saving favorite faculties.

## Features

- **Faculty List**: Displays a list of faculties with essential information.
- **Search Bar**: Allows users to search faculties by title.
- **Category Filter**: Users can filter faculties based on categories.
- **Sorting**: Faculties can be sorted alphabetically by title or category.
- **Detail Page**: Each faculty has a detail page showing extended information.
- **Comparator**: Users can compare two faculties side by side.
- **Favorites System**: Users can save and manage their favorite faculties.

## Technologies Used

- React
- TypeScript
- Bootstrap for styling
